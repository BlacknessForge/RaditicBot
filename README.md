# Raditic Bot

Raditic Bot is a feature-rich, multi-purpose Discord bot engineered to elevate server engagement, streamline community management, and provide robust interactive utilities.

---

## 🚀 Core Features

* **Economy System:** Integrated account registration, daily rewards, secure banking (deposit/withdraw), and active balance tracking.
* **Entertainment Hub:** Interactive text-based games including Coinflip, Slots, Trivia, Wordle, and 2048.
* **Moderation & Server Management:** Comprehensive administrative toolkit featuring `kick`, `ban`, `unban`, `mute`, `unmute`, and bulk message clearing (`purge`) to maintain community safety.
* **Utility & Core Tools:** Low-latency network performance checks (`ping`), server/user lookup tools, and specialized informational commands.
* **Automated Tracking:** Live event handlers (e.g., `presenceUpdate.js`) to monitor member activity and status dynamics in real-time.

---

## 🛠️ Deployment & Development

This repository is maintained for internal project management and development tracking.

* **Cloning the Project:**
```bash
git clone [https://github.com/BlacknessForge/Raditic-Bot-Server.git](https://github.com/BlacknessForge/Raditic-Bot-Server.git)
```
* **Environment Setup:**
```bash
npm install
```
* **Execution:**
```bash
node index.js
```
